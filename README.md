# 二升三暑期學習計畫

這是一個可直接部署的靜態網站，包含：

- 兩個月、8 週暑期日程
- 每日國語隨機生字、造詞、造句、閱讀題
- 每日數學隨機計算題、應用題、訂正區
- 平板練習與作業本搭配安排

## 本機開啟

直接打開 `index.html` 即可使用。

## 雲端部署

可部署到任何靜態網站服務，例如 GitHub Pages、Netlify、Vercel、Cloudflare Pages。

最簡單的方式：

1. 將本資料夾上傳到 GitHub repository。
2. 到 repository 的 Settings > Pages。
3. Source 選擇 main branch。
4. Folder 選擇 `/root` 或放在 docs 後選 `/docs`。
5. 儲存後即可取得網址。

如果使用 Netlify 或 Vercel，直接拖曳這個資料夾或匯入 GitHub repository 即可。
